# Project Memory

这里保存进 Git 的项目共享 memory，给 Claude、Codex 和其他 clone 复用。

## 入口

- [current/INDEX.md](current/INDEX.md)
- [history/README.md](history/README.md)

## 边界

- `memory/current/` 放当前有效的短摘要、恢复入口和文档导航
- `memory/history/` 放阶段切换后仍有短期追溯价值的旧摘要
- `docs/` 放完整正文、研究、设计和历史材料
- `.orchestration/tasks/` 放任务状态
- `git common dir/<repo-name>/memory/` 只放本机临时记忆
