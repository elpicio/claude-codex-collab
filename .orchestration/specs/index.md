# Shared Specs

这些文档给 Claude Code 和 Codex 提供共享规范入口。

- `project.md`：项目范围、控制面边界、任务目录结构
- `documentation.md`：文档、repo 内 `memory/`、本机 local memory 和任务状态的边界
- `coding.md`：代码与测试约束
- `review.md`：review 和 verify 的判断依据
- `adapters.md`：`.claude/` 和 `.codex/` 作为 adapter 时必须遵守的边界

根目录的 `CLAUDE.md` 和 `AGENTS.md` 都应该把这里当成共享规范入口，而不是在各自目录里维护第二套规则。
